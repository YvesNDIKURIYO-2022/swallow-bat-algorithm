# Swallow-Bat Algorithm (SBA) for Threat-Aware Container Truck Routing

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python 3.8+](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/downloads/)

Official implementation of **"A Bio-Inspired Swallow-Bat Algorithm for Threat-Avoidance Routing in the Container Truck Routing Problem"**.

The Swallow-Bat Algorithm (SBA) is a novel hybrid metaheuristic that generates cost-efficient and threat-resilient routes for logistics operations in high-risk environments. It uniquely combines swallow flocking for global exploration with bat echolocation for local exploitation, guided by a proactive threat-aware evasion operator.
